# Basic Arithmetic Problems

---

## **Hard Problems**

### 1. Multiply, Subtract, and Square  
**Description**:  
Write a program that accepts two integer inputs, **x** and **y**. Multiply **x** and **y**, then subtract 15, and finally, square the result.  

**Sample Output 1**:  
Enter x: 5  
Enter y: 3  
Result: 64  

**Sample Output 2**:  
Enter x: 7  
Enter y: 6  
Result: 144

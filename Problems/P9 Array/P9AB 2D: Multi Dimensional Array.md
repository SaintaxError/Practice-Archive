# Multi Dimensional Array Problems
---

### **Problem 1: Sum of All Elements**

**Description:** Given a 2D array of integers, calculate the sum of all elements.

**Sample Input:**  
`arr = [[1, 2], [3, 4], [5, 6]]`

**Sample Output 1:**  
`Sum: 21`

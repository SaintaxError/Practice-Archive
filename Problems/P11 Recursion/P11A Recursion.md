# Recursion Problems

### **1. Sum of N Natural Numbers**  
**Description:** Write a recursive function to compute the sum of the first `n` natural numbers.  
**Sample Runs:**  
```
Enter a number: 5  
Sum: 15  
```  
```
Enter a number: 10  
Sum: 55  
```
